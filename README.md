# RAG Sissoko - Проект RAG системы для работы с PDF

Этот проект представляет собой реализацию RAG (Retrieval-Augmented Generation) системы для обработки PDF-документов. Основная цель — извлечение, разбиение на чанки и подготовка текста для дальнейшего использования в генеративных моделях.

## Компоненты системы

### Основной модуль: `text_splitter.py`

Ключевой компонент системы — класс `TextChunker`, который выполняет интеллектуальное разбиение документов на оптимизированные для RAG фрагменты (чанки).

**Особенности работы `TextChunker`:**
- Использует `RecursiveCharacterTextSplitter` из библиотеки LangChain
- Поддерживает настройку размера чанка и перекрытия
- Сохраняет метаданные оригинального документа для каждого чанка
- Использует умные разделители для сохранения смысловой целостности текста
